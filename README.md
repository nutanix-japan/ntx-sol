## Quick start

Clone this repo

```bash
git clone https://github.com/nutanix-japan/ntx-sol.git
cd ntx-sol/
```

Create python venv

```bash
python -m venv ntx-sol
source ntx-sol/ntx-sol/bin/activate
```

Material for MkDocs can be installed with `pip`:

``` sh
pip install mkdocs-material
```

Serve a development site 

```bash
mkdocs serve --livereload # dirty reload to only reload pages that you are working on
mkdocs serve              # reload all pages in the site
mkdocs serve -a localhost:9999 # run on a particular port

```
